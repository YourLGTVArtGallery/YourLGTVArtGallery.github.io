# Security Policy

TV Art has no backend server — your photos are fetched directly from your storage provider to your TV, and any account tokens are stored only on the TV itself. But if you find a security issue (in the app, this website, or how account access is handled), we want to know.

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.** This gives potential attackers a head start before a fix is available.

Instead, report it privately using one of these:

- [Open a private security advisory](https://github.com/YourLGTVArtGallery/YourLGTVArtGallery.github.io/security/advisories/new) on GitHub (preferred)
- Email [support@xor0game.com](mailto:support@xor0game.com)

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce it
- Your LG TV model and webOS version, if relevant

## What to expect

This is a passion project maintained in spare time, so response times vary — but we take security reports seriously and will do our best to confirm receipt quickly and get a fix out as soon as we can.
